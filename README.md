# ScreenshotMaxxing

ScreenshotMaxxing is a native macOS screenshot utility. It lives in the menu bar, supports keyboard shortcuts, captures areas, windows, and fullscreen screenshots, opens screenshots in a lightweight editor, supports adjustable blur-based obscuration, and stores captures locally.

## Distribution

Official builds are only the signed and notarized DMGs published by Ben Kramer from this repository. Builds from forks, local source checkouts, or other distribution channels are unofficial and may have different code signing, notarization, update, or bundle identity behavior.
